Exercice 1: Create a basic HTML page includes the following elements:

  • HTML and body tags
  • A title for the page
  • A header h1 with a heading
  • A paragraph p with some text
  • An ordered list ol with three list items li
  • An image img with a source (any image URL)
  • a link a to an external website

Exercice 2: HTML forms, create an HTML form that includes the following elements:

  • A form tag with a "POST" method
  • Input a fields for a name, email and password
  • Radio buttons for gender (male, female, other)
  • A dropdown (select) for selecting a country
  • Checkboxes for selecting a country
  • A submit button to submit the form

Exercice 3: HTML tables

Create an HTML table that represents a simple timetable for a week. Include days of the week as headers and fill in some sample data (e.g., time slots and activities) for each day.

Exercice 4: HTML semantic elements

Creat a webpage with semantic HTML5 elements. Use elements such as header, nav, main, article, section and footer to structure your content appropriately.

Exercice 5: HTML forms and validation

Enhance the HTML form from exercise 2 by adding form validation. Make sure that the name and email field must contain a valid address. Display appropriate error messages if the form is submitted incorrectly.

Exercice 6: HTML media embedding

Embed a Youtube video on a webpage using an iframe element. Customize the video size and add a brief description or caption.

Exercice 7: HTML lists and links

Create an HTML page that includes an ordered list of your top 5 favorite book and moovies. Each list item should be a link to more information about the book or movie on an external website.

Exercice 8: HTML CSS integration

Take one of the HTML page you've created and add CSS to style it. You can style fonts, colors, backgrounds and borders to make the page visually appealing.

Exercice 9: HTML forms with JavaScript

Enhance the HTML form from Exercise 2 by adding JavaScript to perform some client-side validation. For example, you can check if the password meets specific criteria or provide real-time feedback when the user enters data.
